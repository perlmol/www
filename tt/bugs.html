PerlMol - Bugs and Requests

Please use SourceForge.net's Tracker to submit bug reports, patches, and feature requests.

Please, if you are reporting a bug, please make sure you include the following:

  • Any and all error messages.
  • Your perl version and platform (the output of perl -V)
  • The version of every PerlMol module involved in the problem.
  • The smallest code sample that can reproduce the problem.
  • The smallest input file (if needed) that can reproduce the problem.
